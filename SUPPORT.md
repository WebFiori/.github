# Support

## Getting Help

If you need help with WebFiori Framework, here are your options:

### GitHub Discussions

For questions, ideas, and community support, please use [GitHub Discussions](https://github.com/WebFiori/framework/discussions).

### Documentation

Check the [official documentation](https://webfiori.com/docs) for guides and API reference.

### Reporting Issues

If you've found a bug or want to request a feature, please open an issue in the appropriate repository:

- **[Framework Core](https://github.com/WebFiori/framework/issues/new/choose)** - Core framework issues
- **[HTTP](https://github.com/WebFiori/http/issues/new/choose)** - HTTP requests, responses, routing
- **[Cache](https://github.com/WebFiori/cache/issues/new/choose)** - Caching functionality
- **[File](https://github.com/WebFiori/file/issues/new/choose)** - File handling and uploads
- **[Json](https://github.com/WebFiori/json/issues/new/choose)** - JSON operations
- **[UI](https://github.com/WebFiori/ui/issues/new/choose)** - DOM manipulation and UI components
- **[Collections](https://github.com/WebFiori/collections/issues/new/choose)** - Data structures and collections
- **[Database](https://github.com/WebFiori/database/issues/new/choose)** - Database operations and queries
- **[CLI](https://github.com/WebFiori/cli/issues/new/choose)** - Command-line interface
- **[Mailer](https://github.com/WebFiori/mail/issues/new/choose)** - Email sending
- **[Errors and Exceptions Handler](https://github.com/WebFiori/err/issues/new/choose)** - Error handling

## Response Time

This is a community-driven project. Response times may vary depending on maintainer availability. We appreciate your patience.
